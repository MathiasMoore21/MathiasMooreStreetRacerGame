# Street Racer - Urban Runner Game


An exciting urban runner game built in Java where players navigate through city streets, dodging obstacles and collecting power-ups to achieve the highest score possible.

##  Game Overview

Street Racer is an endless runner-style game set in a vibrant urban environment. Players control a runner who must survive as long as possible while navigating through procedurally generated obstacles. The longer you survive, the higher your score!


## How to Download and Run

### Prerequisites
- **Java JDK 8 or higher** (Download from [oracle.com](https://www.oracle.com/java/))
- **Any Java IDE** (Eclipse, IntelliJ, NetBeans, VS Code, or BlueJ)

## Installation Steps

### Option 1: Direct Download (Easiest)
1. Click the green **"Code"** button at the top of this repository
2. Select **"Download ZIP"**
3. Extract the ZIP file to a folder on your computer
4. Open your Java IDE
5. Import the folder as a Java project
6. Run the main game class
